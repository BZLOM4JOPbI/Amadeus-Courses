Первое погружение в JavaScript

Изначально JavaScript был создан, чтобы «сделать веб-страницы живыми».

Программы на этом языке называются скриптами. Они могут встраиваться в HTML и выполняться автоматически при загрузке веб-страницы.

JavaScript невероятно универсален и дружелюбен к новичкам. Тем не менее, с JavaScript немного более сложно освоиться, чем с HTML и CSS. Вам придётся начать с малого, продолжая изучение небольшими шагами. Для начала мы покажем вам некоторые основы JavaScript, чтобы создать в консоли вывод "Hello, World!".

Задание
console.log();

Функция console.log используется для вывода любого сообщения, которое необходимо отобразить. Эта функция дает вам доступ к консоли браузера, что позволяет вам выводить строки, массивы и объекты, которые помогают отлаживать код.

console.log('Привет, мир!'); // Выведет строку внутри ковычек

По традиции изучение нового языка программирования следует начинать с 'Hello, World!'.

Продублируйте код снизу в редактор кода. Использование одинарных и двойных ковычек равнозначно, но проверка задания чувствительная к регистру и лишним пробелам, так что набирать следует символ в символ. После отправьте решение на проверку. Строка в ковычках появится в качетсве вывода в консоли.

console.log('Hello, World!');
