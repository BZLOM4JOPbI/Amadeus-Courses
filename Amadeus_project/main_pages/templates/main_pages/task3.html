Переменные в JavaScript

Переменные являются именованными значениями и могут хранить любой тип данных JavaScript.

Вот мы объявляем переменные:

let x = 'Строка';
const n = 5;

Вот что происходит в этом примере:

  • let и const – это ключевые слова, которые говорят JS об объявлении переменной
  • x и n – некоторые имена переменных
  • ' = ' – так называемый оператор присваивания, сообщающий, что далее идет значение переменной
  • 'Строка' и число 5 – значения, которые мы хотим сохранить

После объявления, вы можете ссылаться на переменную по ее имени в вашем коде.

let greetings = 'Привет, Мир!';
console.log(greetings); // Выведет в консоль значение переменной let n = 5;
console.log(n + 5); // Выведет в консоль число 10 (результат сложения)

Возможно ссылаться на переменную даже в объявлении других переменных.

let n = 5;
let x = n + 5;
console.log(x) // Выведет число 10

Несколько правил именования переменных:

  • Первым символом имени может быть буква, нижнее подчеркивание(_) и знак доллара($)
  • После первого символа можно также использовать цифры
  • Именна переменных чувствительны к регистру

Так выглядят недопустимые имена перменных. Попробуйте определить, что не так с каждым из них:

let -var = 'Hello';
let 2name = 'Ivan and Dima';
let @btnText = 'Click';
let procent% = 55;
