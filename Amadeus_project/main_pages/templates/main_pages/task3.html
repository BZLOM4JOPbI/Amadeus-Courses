Операторы в JavaScript

Операторы – это символы между значениями, которые позволяют выполнять различные операции, такие как сложение, вычитание, умножение и другие. В JavaScript есть десятки операторов, поэтому давайте сосредоточимся на тех, которые вы, вероятно, будете использовать чаще всего.

Оператор ' + ' складывает два числа.

console.log(1 + 2); // Выведет число 3

Оператор ' - ' вычитает одно число из другого.

console.log(30 - 10); // Выведет число 20

Оператор ' * ' умножает два числа. Обратите внимание, что это звездочка, а не символ обычно используемый в математике.

console.log(4 * 6); // Выведет число 24

Оператор ' / ' делит одно число на другое. Обратите внимание, что это косая черта, а не символ обычно используемый в математике.

console.log(32 / 4) // Выведет число 8

Выражения в программировании выполняются, следуя приоритету операторов. Из приорите получается, что в примере снизу сначала выполняется умножение между двумя последними числами, хотя сложение записано первым.

console.log(25 - 5 * 2) // Выведет число 15, а не 40

Для большего контроля над вычислениями используется группировка выражений.

Оператор ' () ' группирует выражения, изменяя приоритет операторов. Добавление оператора группировки к предыдущему примеру приводит к тому, что сначала вычисляется 25 - 5.

console.log((25 - 5) * 2) // Теперь выведет число 40

Оператор ' + ' при применении к строкам менят свое поведение и конкатенирует их. Другими словами, он может "склеивать" их.

console.log('Front' + 'end') // Выведет строку 'Frontend'

Символы пробела также нужно учитывать

console.log('Доброе' + ' ' 'утро') // Выведет 'Доброе утро'

Оператор ' = ' присваивает значения переменным.

let option = 'Font size';
